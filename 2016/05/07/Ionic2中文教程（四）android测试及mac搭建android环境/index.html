第四步 安装android SDK

到android官网下载ADT
鉴于国内网络问题可能打不开（如使用vpn或者可以打开请忽略），介绍一下如何在国内下载安装。
在http://www.androiddevtools.cn/下面找到ADT Bundle，选择mac OSX64位，下载到你的mac上。
下载后 找到ADT里面的android/tools目录，运行android文件，将会打开android SDK Manager

然后依次选择：【Android SDK Manager】->【Preferences】会打开【Android SDK Manager - Settings】
窗口；
因为国内网络原因不能直接下载，按照此教程配置设置完成后单击『Close』按钮关闭『Android SDK Manager - Settings』窗口返回到主界面；
现在我们还不知道需要下载哪些包，没关系，先往下进行。

我们进入项目目录再次尝试android调试命令

这时ionic 给出了我们需要的android安装包：

  1. “SDK Platform” for android-23 (Android 6.0 (API 23)下面的所有安装包)
  2. “Android SDK Platform-tools (latest)
  3. “Android SDK Build-tools” (latest)
    打开android SDK Manager，勾选上面所列的安装包，选择安装：install packeges。
    安装时间可能比较长，完成后我们的android环境就搭建完成了!

第六步 连接android设备

将自己的android设备连接到mac上，注意你的android设备要开启USB调试！！然后设备是否可用可以用adb devices查看：

我已经将我的魅族mx4连接到mac上了，但是这里没有显示，说明还不可用。

选择mac 的 关于本机->更多信息-> 系统报告->找到usb选项，右边会出现一系列和usb相关的设备我们找到手机设备.
然后将设备的厂家ID输入到 adb_usb的初始配置文件中，重启服务。

此时就有我们连接的设备了，如果还没有就重新连接一下USB设备或者重启一下mac。
